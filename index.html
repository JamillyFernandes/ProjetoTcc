Arquitetura



Este projeto tem como objetivo desenvolver uma arquitetura serverless para a startup Nova Tech, que está criando um e-commerce e busca aprimorar sua infraestrutura. O foco será garantir alta disponibilidade, escalabilidade e segurança para o ambiente de e-commerce, aproveitando as melhores práticas da AWS e adotando soluções totalmente gerenciadas para simplificar a operação e reduzir custos.

Serverless: Refere-se ao uso de serviços gerenciados que eliminam a necessidade de gerenciar servidores diretamente, como AWS Lambda, API Gateway e DynamoDB. Isso permite que a equipe da Nova Tech concentre seus esforços no desenvolvimento do negócio, enquanto a AWS cuida da infraestrutura. Benefícios: Uma arquitetura serverless oferece escalabilidade automática, custos alinhados ao uso e maior agilidade no desenvolvimento, fatores essenciais para o sucesso de um e-commerce em crescimento.

● Objetivos: Implementar uma infraestrutura serverless na AWS seguindo boas práticas de segurança, escalabilidade e alta disponibilidade.

● Visão Geral dos Serviços Utilizados:

1. Serverless:

AWS Lambda: O coração da lógica de negócios. Cada função Lambda tem responsabilidades específicas, como login, operações administrativas e processamento de compras. Amazon DynamoDB: Armazena dados críticos, como informações de transações e produtos, com baixa latência e alta disponibilidade. AWS Step Functions: Orquestra os fluxos de trabalho para validação de pedidos, processamento de pagamentos, atualização de estoque e notificações. Amazon SQS: Garante o desacoplamento entre componentes, permitindo o envio de mensagens assíncronas.

2. Segurança:

AWS WAF: Protege contra ataques de injeção SQL, Cross-Site Scripting (XSS) e outros padrões de tráfego malicioso. AWS Shield: Fornece proteção automática contra ataques DDoS. AWS Certificate Manager (ACM): Gerencia certificados SSL/TLS para comunicação segura.

3. Armazenamento:

Amazon S3: Armazena conteúdos estáticos e arquivos necessários para o front-end.

4. Rede e Gerenciamento de Tráfego:

Amazon Route 53: Fornece um serviço de DNS escalável e confiável para roteamento de usuários até o endpoint do CloudFront. Amazon CloudFront: Atua como uma CDN (Content Delivery Network), acelerando o acesso a conteúdos estáticos e dinâmicos e adicionando uma camada extra de segurança. Amazon API Gateway: Gerencia as requisições para os serviços serverless, atuando como ponto de entrada para as APIs RESTful. NAT Gateway: Permite que recursos em sub-redes privadas acessem a internet de forma segura, sem expor seus IPs privados, garantindo comunicação unidirecional e proteção.

5. Gestão de Usuários:

Amazon Cognito: Gerencia a autenticação e autorização de usuários, conectando-se diretamente ao serviço de login.

6. Gerenciamento e Governança:

> Amazon IAM (Identity and Access Management): Gerencia permissões de usuários e acessos entre serviços. AWS SAM (Serverless Application Model): Facilita o controle de versão e a implantação do código das funções Lambda.

7. Monitoramento e Observabilidade:

AWS CloudWatch: Centraliza métricas e logs, configurando alarmes para eventos críticos. AWS CloudTrail: Garante auditoria e rastreamento de ações realizadas nos serviços AWS. AWS X-Ray: Realiza rastreamento detalhado de solicitações, ajudando a identificar gargalos.

8. E-mail e Notificações:

Amazon SES: Envia e-mails transacionais, como confirmações de compra e atualizações de pedidos, diretamente para os clientes.

Orçamento



● Estimativa Mensal de Custos:

1. Amazon Route 53: Detalhes: 1 zona hospedada. Custo total estimado: $4,50/mês.

2. Amazon CloudFront: Detalhes: Lida com 10.000 solicitações HTTPS por mês, além de transferir 10 GB de dados de saída para a Internet e 10 GB de dados de saída para a origem mensalmente. Custo total estimado: $1,05/mês.

3. AWS Certificate Manager: Detalhes: Garante certificado SSL/TLS para o CloudFront. Custo total estimado: $0/mês (custo gratuito para certificados públicos).

4. Amazon API Gateway: Detalhes: Gerencia 1 milhão de chamadas mensais. Custo total estimado: $3,50/mês.

5. AWS Lambda: Detalhes: 1 milhão de invocações por mês e 500.000 GB de memória-segundos. Custo total estimado: $4,00/mês.

6. Amazon SQS: Detalhes: 1 milhão de solicitações mensais. Custo total estimado: $0,40/mês.

7. Amazon DynamoDB: Detalhes: Armazena 50 GB de dados, 1 milhão de leituras e 500.000 gravações por mês. Custo total estimado: $4,00/mês.

8. Amazon S3: Detalhes: Armazena 500 GB de dados e 500.000 solicitações de PUT, GET e LIST. Custo total estimado: $11,00/mês.

9. Amazon Cognito: Detalhes: 10.000 MAUs (usuários ativos mensais). Custo total estimado: $2,50/mês.

10. AWS IAM: Detalhes: Sem custos adicionais, pois é gratuito para gerenciar identidades. Custo total estimado: $0/mês.
